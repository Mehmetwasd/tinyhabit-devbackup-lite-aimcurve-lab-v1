# tinyhabit-devbackup-lite-aimcurve-lab-v1
TinyHabit is a lightweight, offline habit tracker designed for simplicity.  / DevBackup Lite is a simple Windows tool that helps developers back up their projects with a single click.  / AimCurve Lab is a small paid utility that visualizes mouse input  and response curves in real time.
